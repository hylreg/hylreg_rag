## [0.1.0] - 2026-03-07

### 新增
- 新增集中式配置模块，支持基于环境变量的 RAG/API 参数配置。
- 为 CLI 和 API 增加向量索引持久化与重载能力。
- 为 API 上传增加防护（文件扩展名与文件大小校验）。
- 增加配置、工具函数、API 行为与 RAG 文档路由的测试套件。
- 增加开源维护资产：
  - 许可证
  - 贡献指南
  - 行为准则
  - 安全策略
  - Issue/PR 模板
  - CI 工作流

### 变更
- 将导入路径重构为基于包的 `src.*` 风格。
- 使用 lifespan 处理器替换已弃用的 FastAPI startup 事件。
- 更新依赖约束以兼容 LangChain。
- 将伪造的 PDF/DOCX 示例生成替换为真实文本样例。
